# Projeto: Controle de Evolução de Paciente
``exemplo de trabalho``

Perguntas:
- Quais são meus pacientes?: 
- Quais os contatos dos meus pacientes?:
- Quantos pacientes eu tenho?:
- Quais pacientes tem x idade?:
- Quais pacientes tem idade maior do que x?:

Dados necessários:

- nome
- telefone
- idade

Funções:

- listar_pacientes()
    lista todos os pacientes
- listar_contato(nome)
    Pesquisa por nome do paciente e imprime a ficha com nome, telfone, idade
- contar_paciente()
    Imprime total de pacientes em meu cadastro
- listar_pacientes_com_idade(idade pesquisada)
    Lista apenas os pacientes que tem idade = x`idade`
- listar_pacientes_com_idade_maior_que_x()
    listar todos os pacientes com idade maior que x `idade`
